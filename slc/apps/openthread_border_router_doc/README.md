## Purpose/Scope

This is an OpenThread Border Router project not intended to be compiled or flashed. Use the pre-built OpenThread Border Router image for your platform instead.

For more information on downloading and flashing the OpenThread Border Router image, see the [OpenThread Border Router Documentation](https://docs.silabs.com/matter/2.8.0/matter-thread/raspi-img).

### SW Requirements

## Steps to Run Demo

1. Download the OpenThread Border Router image for your platform from the Silicon Labs documentation.
2. Flash the image to your device following the steps in the [OpenThread Border Router Documentation](https://docs.silabs.com/matter/2.8.0/matter-thread/raspi-img).

This project is not intended to be built or flashed directly; use the pre-built image instead.
